# customer-churn-prediction

## Overview
Built and evaluated machine learning classification models to predict 
customer churn for a regional gym chain using Google Sheets and SimpleML.

## Dataset
- 100,001 training records
- 10,001 test records
- Features: monthly visits, age, gender

## Models
- Model 1: Full feature set
- Model 2: Reduced feature set

## Results
- Accuracy: 86.41%
- ROC AUC: 0.6396
- Precision-Recall AUC: 0.9123 (non-churn) / 0.2020 (churn)

## Key Finding
Monthly visit frequency was the strongest predictor of churn.

## Business Recommendations
- Visit-based incentive programs for low-frequency members
- Age and gender-targeted promotions to improve engagement

## Tools Used
Google Sheets, SimpleML, Excel
